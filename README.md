## 总结

`Flutter` 相比于 `RN` `Weex` 写 UI 优势比较大，不存在平台不兼容或者丢失某些特性，但是同样的具有跨平台应用平台的通病，涉及到各自平台相关的特性就不得不再去适配各个平台，像是 数据库，地图，webView 等，目前好像是没有较好的解决方法。
